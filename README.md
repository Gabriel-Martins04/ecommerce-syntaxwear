# E-Commerce SyntaxWear

Este projeto é um modelo de página de **e-commerce de calçados** desenvolvido utilizando apenas **HTML e CSS**.
O objetivo do projeto foi praticar a criação de layouts, organização de elementos na página e estilização com CSS.

## 📌 Sobre o projeto

A página simula a interface de uma loja online de calçados, apresentando produtos de forma organizada e visualmente agradável.

Este projeto foi desenvolvido como parte do curso DevQuest, módulo de **desenvolvimento front-end**.

## 🚀 Tecnologias utilizadas

* HTML
* CSS

## 🎯 Objetivos do projeto

* Praticar estruturação de páginas com HTML
* Melhorar habilidades de estilização com CSS
* Criar um layout inspirado em páginas de e-commerce
